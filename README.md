# cacarcade

Couche Architecturale Commune de l'Arcade

# Usage

Copy the `cacarcade` folder to your `include/` folder:

`cp -r /<...>/cacarcade-git/cacarcade /<...>/arcade/include`

On your `arcade`, a `tree include` should look like this:
```
include
`-- cacarcade
    |-- DisplayEventType.hpp
    |-- IDisplayEvent.hpp
    |-- IDisplayModule.hpp
    |-- IGameModule.hpp
    `-- Utils.hpp
```
